Project `4dach` is a Python 3.11 desktop application for roof-plane geometry, layout generation, and reporting, built with PySide6/Qt. Core domain logic lives under `core/` (geometry, layout engine, project state, models, reporting), while GUI logic lives under `ui/`. Tests are in `tests/`. The repo follows stage-based work on non-main branches and uses `uv` for environment and dependency management.
